# Bad Tempeature Converter!

Hello! :D
This was my first C++ project that I have made on 06.11.2025!
I'm trying to learn C++ and decided to upload my projects, so I could follow my own adventure and to prove myself that I am actually making progress.

Project includes Temperature Converter, which works badly, but supports Celsius, Fahrenheit and Kelvins. Other than that project has _test_project_japan.cpp, which I made to look how io.h, studio.h, and fcntl.h work, so I could work with them later! Supports only full values.

Includes test project of Japanese characters, because I wanted to use Unicode symbol for degrees and had to figure out how to do it lol.

## Features

* Supports Kelvin, Celsius and Fahrenheit
* Calculates values for other temperature scales

## Installation

Clone this repository:

```bash
git clone https://github.com/blockersiontko/Bad-TemperatureConverter.git
```

Move into the project directory:

```bash
cd Bad-TemperatureConverter
```
